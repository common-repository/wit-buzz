Your Website Buzz Generator ! Let your users spread the good word

== Description ==

With Wit-Buzz plugin you will collect user Opinion, generate the Buzz and grow your Traffic for free. Measure the user opinion with the 7 Wit-Buzz indicators.
Visit wit-buzz.com to know more about this innovative widget.


== Installation ==

1. Upload `wit-buzz.zip` and unpack to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the Wit-Buzz widget in your sections and sidebars of your templates

== Frequently Asked Questions ==

Do I need to register before using the plug-in?
yes, you have to go to www.wit-buzz.com and ask to register. You will receive an email with all the instructions to paste the code to your web site or be able to use the wordpress plugin as well.

How do I change the look of the icon?
You can change the png file which is stored in the wit-buzz plugin directory. We invite you not to change it as it means to anybody using the service! 

How do I change the size of the icon?
Use the widget's parameter dialog to change the icon size. the default is 30x30 pixels

How do I set the text which is displayed when I pass the mouse over the icon?
Use the widget's parameter dialog to change the text of the link.
